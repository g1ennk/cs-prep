# 말이 트이는 CS 시리즈 

## 1) 컴퓨터 구조
- 데이터와 표현: 비트/바이트, 정수(2의 보수), 부동소수점, 문자(ASCII/Unicode/UTF-8)
- 컴퓨터의 구성: CPU/메모리/I/O, 시스템 버스
- CPU 구성과 기능: ALU, 제어장치, 레지스터, 명령어 처리 흐름
- CPU 동작 원리: 명령어의 형태/오퍼랜드, 인터럽트, 파이프라이닝(해저드 포함), 슈퍼 스칼라
- 주기억장치: RAM, MMU, 캐시(적중률/평균 접근 시간), 지역성/계층 구조
- 보조기억장치: 하드디스크 구조와 접근 시간, RAID
- 입출력 장치: 입출력 모듈, DMA
- [부록] 인터뷰 연습

## 2) 운영체제
- 운영체제란: 정의/기능, 운영체제가 관리하는 하드웨어
- 하드웨어 수행 과정: 커널 모드/사용자 모드, 인터럽트, 시스템 콜, 프로그램 수행 과정
- 프로세스: 문맥/상태, PCB, 스레드와 멀티스레딩
- CPU 스케줄링: 성능 척도, 스케줄링 알고리즘, 문맥 교환
- 프로세스 동기화와 데드락: Race Condition, 크리티컬 섹션, 세마포어/뮤텍스, 데드락
- 메모리 관리 & 가상 메모리: RAM, MMU, 페이징/세그멘테이션, 가상 메모리, 페이지 교체 알고리즘
- 파일 시스템 & 디스크 관리: 파일 보호/할당, 디스크 구성 요소, 디스크 스케줄링
- [부록] 인터뷰 연습

## 3) 네트워크
- 네트워크 기초: LAN/WAN, 전송 방식, OSI 7계층, TCP/IP 계층
- 응용 계층: HTTP, DNS
- 전송 계층: TCP/UDP, 3-Way Handshake, RDT, 흐름 제어/혼잡 제어
- 네트워크 계층: 라우터, IP, 서브넷 마스크, NAT, DHCP
- 데이터 링크 계층: 이더넷, CSMA/CD, 스위치, MAC 주소, ARP
- 네트워크 전체 흐름: 클라이언트–서버 데이터 송수신 과정
- 네트워크 보안과 실무: 보안 3요소, 대칭키/비대칭키, HTTPS, SSL/TLS, 방화벽/VPN/프록시/CDN/CORS
- [부록] 인터뷰 연습

## 4) 데이터베이스
- 데이터베이스 개요: 특성, RDBMS, 데이터베이스 모델링
- SQL 다루기: CRUD, 집계 함수, 서브쿼리, 조인, 페이징, NoSQL
- 정규화: 이상 현상, 1NF~3NF
- 트랜잭션: ACID, 상태/복구, 격리 수준(1~4단계)
- 인덱스 및 데이터 최적화: 인덱스 종류, B-Tree/B+Tree, 복합 인덱스, 샤딩, 파티셔닝
- 데이터베이스 보안: 사용자 권한, SQL 인젝션 방어
- [부록] 인터뷰 연습

## 5) 자료구조
- 자료구조 개요: 분류, 시간/공간 복잡도
- 배열과 리스트: 장단점, 리스트 종류
- 스택과 큐: 스택, 큐, 덱
- 해시 테이블: 해시 함수, 충돌 해결
- 그래프: 종류/구현, 탐색(BFS/DFS)
- 트리: 종류, BST, 트리 순회, AVL, Trie
- 힙: 연산/구현, 우선순위 큐
- [부록] 인터뷰 연습

## 6) 자바와 객체지향(Java & OOP)
- 객체 지향 기본: 객체지향 특징, SOLID
- 자바 개요: 자바 특징, JVM/GC, 참조형 타입, Call by Value, static
- 객체 지향 프로그래밍: OOP, 다형성, 인터페이스/추상 클래스
- 컬렉션: List/Set/Map 비교, Set(HashSet/LinkedHashSet/TreeSet), Map(HashMap/LinkedHashMap/TreeMap/ConcurrentHashMap)
- 멀티 스레딩과 동시성: 구현 방법, synchronized/volatile
- 예외 처리 및 기타: 예외 처리/계층, 직렬화/역직렬화, String vs StringBuilder/StringBuffer
- 모던 자바(Java 8+): 함수형 인터페이스/람다, 컬렉션 vs 스트림
- [부록] 인터뷰 연습
